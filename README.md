# codealpha_tasks
🔹 Task 1: CGPA Calculator

A C++ program that calculates semester GPA and cumulative CGPA by processing multiple semesters, courses, grades, and credit hours. It converts letter grades into grade points, performs accurate computations, and displays structured results with detailed course information.

🔹 Task 2: Login & Registration System

A C++ console-based authentication system that enables user registration and login using file handling. It includes validation for credentials, prevents duplicate usernames, and performs secure verification of stored user data.

🔹 Task 3: Banking Management System

An object-oriented C++ application that simulates core banking operations, including customer creation, account management, deposits, withdrawals, and transaction tracking. It demonstrates OOP principles such as classes, encapsulation, and dynamic memory management.
